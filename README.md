# Slide to Presentation — PoC

This repository holds materials for a **proof of concept (PoC)** that explores AI-assisted, **BCG-compliant** slide production: turning drafts, diagrams, and visual inputs into formatted presentation assets via a **Claude Skill**–style design grammar and pipeline.

**Remote:** [github.com/GCDSDeploy/POC](https://github.com/GCDSDeploy/POC)

## PoC in one line

**Drafts-to-Formatted-PPT** — validate that a machine-readable BCG PPT design grammar plus semantic understanding can drive reliable, on-brand slide generation.
